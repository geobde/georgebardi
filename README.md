# georgebardi.com

- **Framework**: [Astro.js](https://astro.build/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)

## Running Locally

```bash
git clone https://github.com/geobde/georgebardi.git
cd georgebardi
npm install
npm run dev
```
